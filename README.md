# YouTube Content Analyzer & Downloader

## Overview
A machine learning-based web application that analyzes YouTube video metadata and predicts video categories using Natural Language Processing.

## Features
- YouTube video metadata extraction
- Video category prediction
- TF-IDF text vectorization
- Naive Bayes classification model
- Video downloading using yt-dlp
- Interactive dashboard

## Technologies Used
- Python
- Flask
- Machine Learning
- Scikit-learn
- TF-IDF
- Naive Bayes
- yt-dlp

## ML Model
- TF-IDF Vectorizer for text processing
- Naive Bayes classifier for prediction

## Purpose
Built as an academic AI project to explore machine learning applications in real-world content analysis.
